Comment publier
votre article

En quelques étapes simples, votre article sera visible par tous les lecteurs, sur tous les appareils.

1
Créez un compte
Inscrivez-vous avec votre e-mail. Votre profil est synchronisé sur tous vos appareils.
2
Obtenez un code d'accès
Proposez votre idée d'article à la rédaction via le formulaire. Si retenue, vous recevez un code par e-mail.
3
Rédigez votre article
Entrez votre code, choisissez votre rubrique, ajoutez titres, paragraphes, images et citations via l'éditeur de blocs.
4
Publiez en un clic
Votre article est immédiatement visible par tous les lecteurs, partout dans le monde.
